Но тебе всё это не нужно на самом деле! Достаточно после клонирования просто написать git switch origin/feature Одна строчка сделает всё вышеперечисленное. Создаст локально ветку с таким же именем, распакует в неё содержимое внешней ветки и создаст связь между локальной и внешней веткой. Если потом, находясь в этой ветке, выполнишь короткую команду git pull, то увидишь что всё связано и скачивается откуда надо. И даже можно сделать ещё проще. Уже во время клонирования сразу распаковать нужную ветку. git clone --branch feature
